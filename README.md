## 📝 Project Overview
**Naju-Aura** is a premium, high-end landing page designed for luxury fashion and skincare brands. This project focuses on sophisticated visual storytelling, featuring a clean minimalist layout, elegant typography, and a seamless responsive experience. It is built entirely with pure HTML and CSS to demonstrate high-quality UI design skills.

🔗 **Live Demo:** [View Project Live](https://najiba-ta.github.io/Naju-Aura/)

---

## ✨ Key Features
- **Premium UI/UX:** A high-end, luxury aesthetic inspired by modern fashion magazines.
- **Fully Responsive:** Meticulously crafted to look perfect on desktops, tablets, and smartphones.
- **Visual Sections:**
  - **Hero Gallery:** Elegant entrance with a multi-column image showcase.
  - **Aura Fashion:** Beautifully aligned product cards for fashion collections.
  - **Aura Skin Care:** Clean and breathable layout for beauty products.
  - **Visit Our Outlet:** Integrated store information section.
- **Elegant Typography:** Uses professional serif and sans-serif combinations for a refined look.
- **Smooth Layout:** CSS Flexbox and Grid used for a rock-solid, balanced structure.

---

## 🛠️ Built With (Tech Stack)
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom styling featuring:
  - Advanced Flexbox and CSS Grid layouts.
  - Custom color palettes (Gold, Cream, and Dark tones).
  - Hover effects for interactive buttons.
  - Responsive media queries.
- **Google Fonts:** Premium font integration.

---

## 📦 Dependencies
- **Font Awesome:** For elegant social media and contact icons.
- **Pure CSS:** No frameworks like Bootstrap were used, ensuring 100% custom-crafted design and fast performance.
